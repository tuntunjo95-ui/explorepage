# 屏规格 · 星际之旅 UniverseRoute（关系养成体系）

> 来源 `src/popup/screens/UniverseRoute.jsx`。深色星空屏。视觉优先 web-capture `?s=universe&p=hhhhll`。

## 1. 身份与导航
- 入口：私聊右上**土星图标** → 本屏，带 `peer`；我的主页 `🪐 En Route` 也进本屏。
- 出口：顶栏返回 → `chat(peer)`。
- StatusBar：`dark`（深色反白）。

## 2. 结构（三层：固定顶 / 可滚星河 / 固定底）
**固定顶栏**：返回箭头（白）+ 居中 `Universe Route` + 右 `?` 帮助。

**可滚动星河**（从上到下 = 天数从大到小、越远越高）：
- 背景：深紫星空渐变（`#0A0B32→#151A58→#0B0926`）。
- 一条**轨道虚线** uni-route-line 串起 4 个里程碑星球：

| 星球 | 天数 | 进入对数 | 相对大小 |
|---|---|---|---|
| **Eternal Star** | 365 天 | 0 pairs | 最小最远（顶） |
| **Bend Star** | 120 天 | 0 pairs | 大 |
| **Echo Star** | 30 天 | 12,864 pairs | 中 |
| **Flicker Star** | 7 天 | 178,879 pairs | 中（最近） |

- **双人头像舱**（在 Flicker 与地球之间）：me 头像 + peer 头像并排在一个底座上 = 当前这段关系的进度位置。
- 底部**地球地平线**（uni-earth）。

**固定底部**：左 `Day 2 / 7`（当前进度）+ 右 `. Memory ✦ ➤➤`。

## 3. 交互
- **点星球** → 该星球 `active`（高亮/放大），其余复原。默认 Flicker 高亮。
- 星河区域**可纵向滚动**（星球分布在很长的轨道上，初始视口在底部 Flicker/地球附近）。
- 返回 → 回到来源私聊。

> 产品语义（来自 HANDOFF §4）：聊满 20 轮点亮 2 星、50 轮点亮 3 星变灰色星球。星球可点显示「多少对关系进入该阶段」。本屏是**展示+导航**，不含真实进度计算。

## 4. 状态清单
`active`（当前高亮星球 key，默认 `flicker`）。

## 5. 验收点
- [ ] 4 星球按天数从大到小、由高到低、由小到大排布，轨道虚线串联。
- [ ] 点不同星球可切换高亮。
- [ ] 双人头像舱在 Flicker 与地球之间。
- [ ] 顶栏、底部 Day x/7 固定不随星河滚动。
