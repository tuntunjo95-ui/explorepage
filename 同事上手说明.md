# Pop Up 复刻站 · 独立副本（上手说明）

这是一份完整、独立的源码副本，和原作者的网站互不影响。你可以自由修改、单独部署。

## 跑起来（首次）
需要 Node.js 18+（没有就去 nodejs.org 装）。终端进入本文件夹后：

```bash
npm install      # 装依赖（首次约 1-2 分钟）
npm run dev      # 本地预览
```

然后浏览器打开终端里显示的地址（形如 http://127.0.0.1:5173/）。
⚠️ 用 127.0.0.1，不要用 localhost（IPv6 可能拒连）。

## 代码在哪
- 所有页面/交互：`src/popup/screens/*.jsx`（对话列表是 `MessageList.jsx`）
- 样式：`src/popup/popup.css`
- 图片素材：`src/assets/` 和 `public/`
- 路由：`src/App.jsx`（用 ?s=<页面名> 切换，如 ?s=message）

## 打包成静态网站
```bash
npm run build    # 产物在 dist/，可直接上传到任意静态托管
```

## 单独部署成公开链接（可选，二选一）
- 最简单：把 `npm run build` 出来的 `dist/` 文件夹拖到 Cloudflare Pages / Vercel / Netlify 的网页后台。
- 命令行（需自己的 Cloudflare 账号）：
  ```bash
  npx wrangler@latest pages deploy dist --project-name=你自己的项目名
  ```
  ⚠️ 项目名取一个**和原站不同**的名字，这样你的站是独立的公开链接，不会覆盖原站。
